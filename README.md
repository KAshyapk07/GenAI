# Generative AI (Gen AI)

This repository contains all coursework, experiments, and mini-projects related to the **Generative AI** subject.

The focus of this subject is to understand how modern language models work and how they can be used to generate text using prompt engineering and transformer-based architectures.


## Learning Outcome
This repository demonstrates practical understanding of **Generative AI concepts**, limitations of early language models, and hands-on implementation using real code examples.

## Note
This repository is maintained for academic and learning purposes.
